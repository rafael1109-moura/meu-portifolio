Portfólio — Rafael Moura
Meu portfólio pessoal feito em HTML, CSS e JavaScript puro. Sem frameworks, sem dependências, um arquivo só.
Por que um arquivo só?
Queria algo que funcionasse em qualquer lugar sem precisar de npm install, servidor ou build. Abre no navegador e pronto.
O que tem dentro

Apresentação com links para LinkedIn e GitHub
Seção de habilidades com barras de progresso animadas
Timeline de experiência profissional
Formação e certificações
Formulário de contato

Tecnologias

HTML5
CSS3 (variáveis, grid, flexbox, keyframes)
JavaScript vanilla (IntersectionObserver para as animações de scroll)
Google Fonts (Space Mono + DM Sans)

Como rodar
Não tem como errar:
bash# Clona o repositório
git clone https://github.com/rafael1109-moura/portfolio

# Abre o arquivo no navegador
# Windows
start index.html

# Mac
open index.html

# Linux
xdg-open index.html
Ou simplesmente arrasta o arquivo portfolio-rafael.html no navegador.
